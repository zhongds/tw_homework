Two ways to setup the project：

### development environment
```
　　npm install
　　npm start
```

### production environment
```　
 　　npm install
   　npm run build  
 　　npm run deploy:prod
```

open "http://localhost:3000" in browser

#### default port : 3000, you can change the port in package.json file
